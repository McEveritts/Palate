# AetherFlow Design System

## Source
Extracted from `AetherFlow/frontend/src/app/globals.css`.

## Visual Foundations
- **Theme**: Dark mode default (`slate-950` backgrounds, `slate-200` text).
- **Typography**: Inter (`--font-inter`).
- **Accent Color**: Indigo (`indigo-500`, `indigo-400`).
- **Style**: Glassmorphism. Components heavily utilize `backdrop-blur`, semi-transparent white/slate backgrounds, and subtle borders to create depth.
- **Card Patterns**: `.glass-card`, `.glass-panel`, `.glass-button`.
